О программе
____________________________________________________________________________________________
Продукт  VarGenKTO был разработан на языке высокого уровня java в обучающих целях.
Было предусмотрено, что данная программа будет распространяться свободно.
Для написания данной программы была использована библиотека iText — это библиотека классов
для генерации, анализа и изменения документов в формате Portable Document Format (PDF),
 а так же плагин обёртыватель launch4j, для создания ЕХЕ файла. ВСе технические данные были
 получены из ///

Как пользоваться
____________________________________________________________________________________________
Перед запуском программы необходимо наличие java jre 1.8.0_301 или выше,а так же jdk
последней версии.
Для запуска программы пользователю необходимо заупстить файл VarGenKTO.exe, так же
необходимо наличие папки fonts для корректного создания файлов домашних задач (ДЗ).
После запуска пользователю будут предложены три варианта, каждому из которых соответствует
одна из кнопок:
 1) создать файл первых ДЗ, кнопка "ДЗ#1"
 2) создать файл вторых ДЗ, кнопка "ДЗ#2"
 3) создать файл третьих ДЗ, кнопка "ДЗ#3"
При выборе одного из вышеперечисленных пунктов пользователю будет предложено
 а) создать новый файл ДЗ (истинно для ДЗ 1-3), для этого нужно нажать на кнопку
 "Сгенерировать ДЗ #(номер задания)".
 б) создать выборочно один из пунктов ДЗ (истинно для ДЗ 1,2), для этого нужно выбрать один
  или несколько желаемых пунктов, после чего нужно нажать на кнопку
  "Сгенерировать ДЗ #(номер задания)".
 в) создать файл по имеющемуся ключу (истинно для ДЗ 1-3), для этого нужно поместить
  имеющийся ключ, в "Поле для повторной генерации", после чего нужно нажать на кнопку
  "Сгенерировать ДЗ #(номер задания)".
 г) создать файл по имеющемуся ключу с новым выбранным пунктом (истинно для ДЗ 1,2),
 для этого нужно поместить имеющийся ключ, в "Поле для повторной генерации", затем
 выбрать задачи, которые необходимо создать повторно, после чего нужно нажать на кнопку
 "Сгенерировать ДЗ #(номер задания)".
После проделанных действий всплывёт окно-результат, сообщающее исход работы программы,
Чтобы создать новый файл того же ДЗ, сначала необходимо закрыть окно-результат.
Чтобы выбрать другое ДЗ для создания, сначала необходимо закрыть окно выбранного ранее ДЗ.

Возможные ошибки
____________________________________________________________________________________________
При запуске исполняемого файла может возникнуть ошибка сообщающая об отутсвии установленной
java. В этом случае проверьте наличие пути до java в системной переменной Path. В случае
если путь корректный, советуется создать дополнительную переменную JAVA_HOME, в пути
которой прописать путь до недавно установленной jre или jdk.

В случае ошибочной генерации файла, когда создаётся файл наполненный "мусором",
проверьте наличие папки fonts.

Если при повторной генерации вышло сообщение о неверно заданных параметрах, советуется
проверить правильность ключа.
При создании файла, в поле для ключей пробелы должны отсутствовать.
